# 10.-Object-Oriented-Programming-Challenge-SVG-Logo-Maker

## Purpose
The focus of this project is to build a Node.js command-line application that takes in user input to generate a logo and save it as an SVG file.

# Built With
* JavaScript
* HTML
* CSS
* Node.js
* Express.js
